‚ö†Ô∏è Detroit-Specific Home Considerations:

1. Roof Inspection (CRITICAL!)

Detroit's harsh winters with heavy snow and ice dams can damage roofs quickly. Before closing on any home, get a professional roof inspection.

Free comprehensive roof inspection for all new Detroit residents! We'll check:

Schedule Free Inspection
2. Winterization
  • Insulation check (attic should have R-49 minimum)
  • Furnace inspection before winter
  • Seal windows and doors
  • Install storm windows if not present
3. Plumbing
  • Check for lead pipes (common in older Detroit homes)
  • Insulate pipes to prevent freezing
  • Test water pressure
4. Foundation
  • Inspect for cracks (freeze-thaw cycles)
  • Ensure proper drainage away from house
  • Sump pump installation if needed
